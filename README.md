# Aim Trainer

Petit jeu d'entraînement à la visée (aim training) en Python, utilisant Pygame.

## Fonctionnement

Des cibles apparaissent aléatoirement à l'écran et grossissent progressivement.
Cliquez dessus avant qu'elles n'atteignent leur taille maximale. La partie se
termine après un nombre limité d'erreurs (3 vies).

## Installation

```bash
pip install pygame
```

## Lancer le jeu

```bash
python AimTrainerV2.py
```
